# 📊 InsightX

### Offline-First Enterprise Excel Analytics Dashboard

InsightX is a modern, high-performance Excel analytics platform built with **React 19**, **TypeScript**, and **Vite**. It enables users to upload, analyze, transform, visualize, and generate insights from Excel datasets completely inside the browser—without requiring any backend server or cloud infrastructure.

Designed with privacy, speed, and enterprise usability in mind, InsightX performs all processing locally, ensuring complete data security while delivering an interactive analytics experience.

# 🚀 Features

## 🔒 Offline-First & Privacy Focused

InsightX processes every Excel workbook directly inside the browser.

- No backend server
- No API calls
- No cloud storage
- No user tracking
- No cookies

All uploaded files remain completely on the user's device.

---

## 📂 Excel Processing Engine

Powered by **SheetJS (XLSX)** for high-performance workbook parsing.

Supports:

- XLSX
- XLS
- Multiple Worksheets
- Large Datasets
- Dynamic Column Detection

---

## 🧮 Interactive Pivot Builder

Create Excel-style pivot tables with an intuitive interface.

Features include:

- Drag & Drop Configuration
- Rows & Columns
- Values
- Filters
- Sorting
- Expand / Collapse
- Automatic Subtotals
- Export Ready Layouts

---

## 📊 Visualization Center

Build powerful dashboards using multiple chart types.

### Visualization Modes

- Frequency Analysis
- Relationship Analysis

### Supported Charts

- Column Chart
- Bar Chart
- Line Chart
- Area Chart
- Pie Chart
- Donut Chart
- Treemap

Each visualization automatically generates intelligent observations based on the selected dataset.

---

## 📈 Analytics Dashboard

Automatically generates statistical insights including:

- Missing Value Detection
- Duplicate Detection
- Numeric Distribution
- Average & Median
- Minimum & Maximum Values
- Data Type Recognition
- Cardinality Analysis

The dashboard also highlights important trends and warning indicators to support better decision-making.

---

## 🏷 Header Mapper

Rename dataset headers without modifying the original workbook.

Features:

- Safe Column Mapping
- Undo / Redo
- Instant Preview

---

## 🎨 Workspace Personalization

Choose between three professional themes:

- ☀️ Light
- 🌙 Dark Navy
- 🌲 Forest Green

Theme preferences are automatically stored using IndexedDB.

---

## 💾 Local Storage

InsightX uses IndexedDB to preserve:

- Uploaded Workbook
- Active Sheet
- User Preferences
- Theme
- Workspace Settings

Everything remains stored locally inside the browser.

# 💻 Installation

Clone the repository

```bash
git clone https://github.com/yourusername/InsightX.git
```

Move into the project directory

```bash
cd InsightX
```

Install dependencies

```bash
npm install
```

---

# ▶ Running the Application

```bash
npm run dev
```

By default, the application runs at:

```
http://localhost:5174
```

---

# 📦 Production Build

```bash
npm run build
```

Preview the production build:

```bash
npm run preview
```
